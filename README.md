</div>

>

## ⚔️ **기능 소개**
 
</div>

### ⚙️ 장애물 시스템

- 피격될 수 있는 **장애물 오브젝트** 존재:
  - **화살**: 일정 간격으로 발사되어 맞으면 즉사  
  - **도끼**: 회전하면서 주기적으로 지나가는 함정  
- **슈퍼 점프대**: 밟으면 천장에 부딪혀 사망  
- **낚시(트랩) 요소**:
  - “낙원”처럼 보이는 이미지 구역에 접근하면 실제로는 **뒤에 용암이 등장**  
 

<br>

### 💊 아이템 시스템

- **돌멩이** : 장애물의 반응을 유도하거나 타이밍을 계산하는 데 사용  
- **열쇠(Key)** : 특정 구간을 해제하고 이벤트를 트리거  
- **그린애플(Green Apple)** : 일정 시간 던지기 파워 강화 (`throwPower = 100`)  
- **일반 사과(Apple)** : 아드레날린 지속 시간 증가 효과 (예정)

<br>

### ⚡ 아드레날린 모드

- **열쇠 획득 후 4초 뒤 자동 발동**
- 이동 속도(`moveSpeed = 40`), 점프력(`jumpPower = 200`) 대폭 증가  
- **5초 동안 유지 후 자동 복귀**

<br>

### 🧠 상호작용 시스템

- 마우스 중앙 기준으로 **Raycast**를 쏴서 상호작용 가능한 오브젝트 탐지  
- **UIManager**를 통해 오브젝트 설명 UI를 표시/숨김 처리  
- 아이템 종류(`ItemData.displayName`)에 따라 **자동 위치 및 회전 설정**  
- **Action(델리게이트)**를 사용해 특정 이벤트(예: 열쇠 획득 후 효과 발동)를 손쉽게 연계

<br>

### 🧩 기타 시스템

- **UI 순차 등장 시스템** : `Coroutine`으로 1초 간격으로 UI 표시하여 3,2,1 땅! 하고 달리는 경주처럼 UI 연출 시도 
- **물리 기반 던지기 시스템** : `Rigidbody.AddForce()`를 이용한 아이템 던지기  
- **LayerMask 필터링** : 상호작용 가능한 오브젝트만 감지
